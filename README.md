# Business Landing Page with WhatsApp

This project is a simple and clean business landing page made to help small businesses connect with customers through WhatsApp.

It is easy to use, fast, and works well on mobile phones.

## What this landing page includes
- Simple and modern design
- WhatsApp contact button
- Mobile-friendly layout
- Easy to edit HTML and CSS files

## How to use it
1. Open the `index.html` file using a text editor (VS Code is recommended)
2. Change the business name and text to your own
3. Replace the WhatsApp link:
   https://wa.me/CLIENT_NUMBER  
   with your own WhatsApp number  
   (example: https://wa.me/573001234567)
4. Save the file and upload it to your hosting or website

## Who is this for?
- Small business owners
- Entrepreneurs
- Freelancers
- Local services

This landing page was created as a beginner-friendly project focused on simplicity and real use for businesses.
